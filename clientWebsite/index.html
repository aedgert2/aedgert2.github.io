Learn CSS with Angel

Welcome! This site will help you understand the fundamentals of CSS through clear examples, interactive lessons, and fun challenges. Whether you’re brand new or brushing up on your front-end skills, you’ll learn how to style beautiful and responsive web pages.

Start Learning

What You’ll Learn

  • Selectors: Learn how to target and style HTML elements effectively.
  • Colors & Fonts: Master typography, backgrounds, and color schemes.
  • Box Model: Understand margins, borders, and padding for perfect layouts.
  • Layout & Positioning: Explore Flexbox, Grid, and CSS positioning techniques.
  • Animations & Transitions: Add motion and smooth effects to your designs.
  • Practice: Test your knowledge with hands-on exercises.

About This Project

This project was created by Andrew Edgerton as part of a foundational web design course. The goal is to help new learners, like Angel, develop a strong understanding of CSS principles through guided examples and interactivity.

References: W3Schools, MDN Web Docs, CSS-Tricks, Flexbox Froggy.
