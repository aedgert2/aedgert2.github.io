Project Overview

Learn CSS with Angel

Project Description

This web application is an interactive learning website that helps beginners understand the fundamentals of Cascading Style Sheets (CSS). It focuses on visual explanations, interactive examples, and simple exercises to help users like Angel build a strong foundation in front-end web design.

Purpose

The goal is to make CSS learning approachable and practical by providing guided lessons, visuals, and exercises where users can experiment with styling elements directly in the browser.

Intended Users

  • Primary user: Angel – a student learning CSS for the first time.
  • Secondary users: Other students or beginners interested in learning web design.

Overview of Website Content

  • Home Page – overview and purpose of the site
  • Selectors – CSS selectors and specificity
  • Colors and Fonts – styling text, backgrounds, and typography
  • Box Model – spacing, borders, padding, and margins
  • Layout and Positioning – flexbox, grid, and positioning
  • Animations and Transitions – smooth CSS effects
  • Practice – interactive CSS challenges

Client Information

  • Client Name: Angel
  • Organization/Institute: Co-worker, friend, and student at CPCC

Dynamic Functionality

  • Collapsible Lessons: Used across pages to organize long content (inspired by MDN Docs).
  • Interactive Color Picker: Lets user change color values on live examples (used on Colors and Fonts page, inspired by W3Schools Color Picker).
  • Box Model Visualizer: Interactive display showing margin, padding, and border spacing (used on Box Model page, inspired by CodePen visualizers).
  • Flexbox/Grid Live Demo: User adjusts element alignment interactively (Layout and Positioning page, inspired by Flexbox Froggy).
  • Animation Speed Slider: Adjust transition timing for demos (Animations page, inspired by Animista).
  • Interactive Quiz: Checks user understanding with instant feedback (Practice page, inspired by W3Schools CSS Quiz).
  • AJAX Lesson Loader: Dynamically loads example snippets without refreshing the page (Practice page, uses jQuery .load()).
